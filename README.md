### В наш пакет дополнений входит ROS ноды для обнаружения окон, автономной навигации по стене, посадка на зарядную станцию и другие мелкие фишки для нашего проекта.

### Our add-on package includes ROS nodes for window detection, autonomous wall navigation, charging station landing, and other small features for our project.
